## LED ON/OFF Control using ESP8266 and Blynk Cloud

## 📌 Project Overview
The LED ON/OFF Control using ESP8266 and Blynk Cloud project is an IoT application that allows users to control an LED remotely using the Blynk IoT platform. The ESP8266 (NodeMCU) connects to Wi-Fi and communicates with the Blynk Cloud, enabling users to switch the LED ON or OFF from the Blynk mobile app or web dashboard.

## 🎯 Objective
Control an LED remotely using the Blynk Cloud.
Connect the ESP8266 to a Wi-Fi network.
Learn IoT-based remote device control.
Understand virtual pins and Blynk widgets.

## ⚙️ Working Principle
The ESP8266 connects to the Wi-Fi network.
The ESP8266 establishes a connection with the Blynk Cloud.
A Button widget in the Blynk app sends ON/OFF commands through Virtual Pin V0.
The ESP8266 receives the command.
The LED turns ON or OFF based on the button state.

## ▶️ How to Run
Connect the LED to the ESP8266.
Create a new Template in the Blynk Cloud.
Create a Device from the template.
Add a Button Widget and connect it to Virtual Pin V0.
Copy the Template ID, Template Name, and Auth Token into the Arduino code.
Enter your Wi-Fi SSID and Password.
Upload the code using the Arduino IDE.
Open the Blynk app or web dashboard.
Press the button to turn the LED ON or OFF.
